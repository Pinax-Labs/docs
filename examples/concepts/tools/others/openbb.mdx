## Code

```python cookbook/tools/openbb_tools.py
from pinaxai.agent import Agent
from pinaxai.tools.openbb import OpenBBTools

agent = Agent(
    tools=[OpenBBTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("Get the latest stock price for AAPL")
```
