## Code

```python cookbook/tools/sleep_tools.py
from pinaxai.agent import Agent
from pinaxai.tools.sleep import SleepTools

agent = Agent(
    tools=[SleepTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("Wait for 5 seconds before continuing")
```
