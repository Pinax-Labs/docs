### Authenticate with pinaxai

Authenticate with pinax.tech by running the following command:

```shell
pinaxai setup
```

or by exporting the `PINAX_API_KEY` for your workspace from [app.pinax.tech](https://app.pinax.tech/settings)

<CodeGroup>
```bash Mac
export PINAX_API_KEY=ag-***
```

```bash Windows
setx PINAX_API_KEY ag-***
```
</CodeGroup>
