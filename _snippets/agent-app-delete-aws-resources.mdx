## Delete AWS resources

Play around and then delete AWS resources using:

<CodeGroup>

```bash terminal
pinaxai ws down --env prd --infra aws
```

```bash shorthand
pinaxai ws down prd:aws
```

</CodeGroup>

or delete individual resource groups using:

<CodeGroup>

```bash app
pinaxai ws down --env prd --infra aws --group app
```

```bash api
pinaxai ws down --env prd --infra aws --group api
```

```bash database
pinaxai ws down --env prd --infra aws --group db
```

</CodeGroup>
